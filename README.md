##  Retail Sales Data Analyst Project

##  Project Overview
A complete intermediate-level Data Analyst project 
built using Excel, SQL, and Power BI.

##  Tools Used
- **Microsoft Excel** — PivotTables, Charts, Slicers, Formulas
- **SQL (SQLite)** — Aggregation, CTEs, Window Functions, Subqueries
- **Power BI** — DAX Measures, Interactive Dashboard
- **HTML/CSS/JS** — Custom Web Dashboard

##  Files
| File | Description |
|------|-------------|
| `retail_sales_analysis.xlsx` | Excel workbook with 500 rows of sales data |
| `sales_data.csv` | Raw dataset used for SQL analysis |
| `retail_analysis_queries.sql` | 7+ SQL queries from basic to advanced |
| `my_sales_dashboard.html` | Interactive HTML dashboard |
